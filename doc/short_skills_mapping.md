# Danh Sách Skills Rút Gọn (Short Aliases)

Theo yêu cầu "tên skill viết tắt chữ cái đầu", dưới đây là bảng ánh xạ bộ skills của dự án:

| Viết Tắt | Tên Gốc (Original) | Ý Nghĩa (Quy luật) | Mô Tả Ngắn |
| :--- | :--- | :--- | :--- |
| **`b`** | `brainstorming` | **B**rainstorming | Lên ý tưởng, thảo luận. |
| **`cc`** | `clean-code` | **C**lean **C**ode | Chuẩn hóa code. |
| **`a`** | `architecture` | **A**rchitecture | Kiến trúc hệ thống. |
| **`sa`** | `security-audit` | **S**ecurity **A**udit | Kiểm tra bảo mật. |
| **`pe`** | `prompt-engineer`| **P**rompt **E**ngineer| Tối ưu hóa prompt AI. |
| **`gc`** | `git-commit` | **G**it **C**ommit | Tạo commit message. |
| **`rg`** | `readme-generator`| **R**eadme **G**enerator| Viết tài liệu README. |

## Ghi Chú
- Các skill rút gọn này hoạt động giống hệt skill gốc.
- Bạn có thể gọi nhanh bằng cách gõ `@b`, `@cc`, `@a`, v.v.
