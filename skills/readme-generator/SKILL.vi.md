# README Generator

This skill generates professional README files.
**Mục tiêu**: Tạo file README chuyên nghiệp.

## When to use
- User wants to create or update `README.md`.
- User asks for "project documentation".

## Structure
1.  **Project Title & Overview**: What it is, who it's for.
2.  **Key Features**: Bullet points.
3.  **Tech Stack**: Languages, frameworks, tools.
4.  **Prerequisites**: What is needed to run it.
5.  **Getting Started**: Step-by-step installation and run guide.
6.  **Architecture**: High-level design (optional but recommended).
7.  **Environment Variables**: Config reference.
8.  **Deployment**: How to deploy.
9.  **Troubleshooting**: Common issues.

Instructions:
- Analyze the codebase to fill in the details.
- Use clear, professional Markdown formatting.
