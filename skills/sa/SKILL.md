# Security Audit Skill

This skill performs a security audit, derived from the `security-auditor` skill, tailored for: **Kiểm tra lỗ hổng bảo mật**.

## Use this skill when:
- Running security audits or risk assessments (Kiểm tra bảo mật và đánh giá rủi ro)
- Investigating vulnerabilities (Điều tra lỗ hổng)
- Validating security controls (Xác thực kiểm soát bảo mật)

## Instructions
1.  **Assess**: Identify the target assets and scope.
2.  **Scan**: Use available tools (if configured) or manual analysis to find vulnerabilities.
3.  **Report**: Document findings with severity and remediation steps.
